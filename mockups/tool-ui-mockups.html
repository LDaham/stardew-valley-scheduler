A 좌측 목록 + 우측 상세 (고정)

왼쪽에 NPC 목록을 고정하고 오른쪽에 상세를 표시. 클릭하면 즉시 전환(뒤로가기 불필요). 상단 검색으로 빠르게 찾기. 앱의 좌측 사이드바 패턴과 일관.

C 전체 펼쳐 보기 + 빠른 이동
